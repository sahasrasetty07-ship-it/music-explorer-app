Music Recommendation App 
Project Description:
This project is a web application that allows users to search for songs and discover similar tracks. It uses the Last.fm API to fetch music data such as song names, artist information, and related songs.
Purpose:
The purpose of this project is to learn how to use public APIs and build an interactive web application using HTML, CSS, and JavaScript.
API Used:
Last.fm API
Features:
* Search for songs
* Display song and artist information
* Show similar or recommended songs
* Basic sorting/filtering of results
Technologies Used:
* HTML
* CSS
* JavaScript
